# convertfs
**convertfs** is a file converter (packed and unpacked eigenstrat <-> binary and plain plink).

```
git clone https://github.com/stasundr/go-convertfs.git convertfs
cd convertfs
go build
```

or

```
git clone https://github.com/stasundr/go-convertfs.git convertfs
cd convertfs
go run main.go
```
